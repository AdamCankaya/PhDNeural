# PhD Projects Repository

This repository contains multiple projects related to PhD research. Each project is contained within its own dedicated directory.

## Projects

### [Bio-NAS](./Bio-NAS)
**Path:** `Bio-NAS/`

A Spatio-Temporal Bio-NAS framework investigating whether Biologically-Informed Neural Architecture Search (where neural pathways are constrained by known human anatomy, e.g., Gene Regulatory Networks) outperforms unconstrained, mathematical optimization in multi-omic disease prediction. 

Features:
- **Track A (Control):** Unconstrained standard NAS optimizing for mathematical efficiency.
- **Track B (Bio-NAS):** Biologically constrained neural architectures using KEGG/Reactome blueprints.
- Comparative A/B testing initially focused on Breast Invasive Carcinoma (BRCA) before scaling to four other pathologies.

For detailed documentation, tracking, and the codebase, please see the [Bio-NAS README](./Bio-NAS/README.md).
