PhD Research Portfolio

Welcome to my PhD research projects.

Bio-NAS

Objective: Investigate whether Biologically-Informed Neural Architecture Search (where neural pathways are constrained by known human anatomy, e.g., Gene Regulatory Networks) outperforms unconstrained, mathematical optimization in multi-omic disease prediction.

Planned Input Data: Multi-omic data (initially Breast Invasive Carcinoma - BRCA).

Planned Final Output: Optimized neural architecture comparison (A/B testing) for disease prediction.

Epigenetic Deconvolution SaaS (β-VAE)

Objective: Build a machine learning platform capable of mathematically unmixing bulk epigenetic data into single-cell resolution.

Planned Input Data: Bulk epigenetic data.

Planned Final Output: Scalable, secure enterprise API platform for epigenetic deconvolution.
